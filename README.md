### 팀원 전공/학번/이름/github 계정 

1. 개요 : 어떤 소프트웨어인지 대략적인 소개
2. 라이선스 : 회사 및 가격(예: 오토캐드 250만원/연) 또는 오픈소스(예: Apach 2.0) 등 
3. 주요기능 
   - 팀원별 1개 기능 이상 작성할 것
